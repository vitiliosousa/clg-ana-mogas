Sistemas de Informação (SI)

É uma área da computação focada na integração entre tecnologia, processos organizacionais e pessoas, com o objetivo de gerenciar e melhorar o fluxo de informações em empresas e organizações.

Principais Componentes de um Sistema de Computação

Hardware – Parte física

  • Processador – Executa as instruções.
  • Memória (RAM, ROM) – Armazena dados temporários e permanentes.
  • Dispositivos de entrada/saída – Teclado, mouse, monitor, impressora.
  • Dispositivos de armazenamento – HDs, SSDs, pen drives.

Software – Parte lógica

  • Sistema Operacional (SO) – Interface entre hardware e usuário (ex: Windows, Linux).
  • Aplicativos – Programas usados pelo usuário (ex: navegadores, editores de texto).
  • Drivers – Permitem que o SO se comunique com o hardware.
  • Usuário – Pessoa que interage com o sistema por meio do software.

Funcionamento Básico

  • Entrada de dados (ex: digitar no teclado)
  • Processamento (realizado pela CPU com apoio da memória)
  • Armazenamento (salvar arquivos, por exemplo)
  • Saída de dados (ex: exibir no monitor)

Servidores

  • Dispositivos móveis
  • Computadores pessoais
